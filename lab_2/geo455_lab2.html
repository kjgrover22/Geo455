What's it all about?

This is the final result of the Leaflet Quick Start Guide. The map demonstrates Leaflet basics, such as setting up a Leaflet map, and adding markers and popups. I used it to show one of my favorite landscapes in Wisconsin.

The placement and styling of the title and description boxes is done using CSS. Check out the source code of this page to see how.
